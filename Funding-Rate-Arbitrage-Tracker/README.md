# Funding-Rate-Arbitrage-Tracker

## Project Overview

This project is a beginner-friendly but portfolio-ready ML and Quant Development system built with Python.

## Features

- Professional dark-themed visualization
- Synthetic fallback data support
- Beginner-friendly architecture
- Quant research workflow
- Independent execution
